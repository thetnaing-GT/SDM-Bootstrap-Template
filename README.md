## Set up Project

### 1. Install PHP Dependencies

```bash
composer install
```

### 2. Install NodeJS Dependencies

```bash
npm install
```

### 3. Set Up Environment File

```bash
cp .env.example .env
```

### 4. Generate Application Key

```bash
php artisan key:generate
```

### 5. Run Frontend Assets

```bash
npm run dev
```

### 6. Start the Development Server

```bash
php artisan serve
```
